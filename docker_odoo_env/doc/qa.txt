Quality Assurance
-----------------

Corre los tests del modulo testeado por defecto

Para poner un modulo por defecto hacer

oe qa -m modulo_a_testear

opciones:

--create-database
    Crea una base de datos vacia con el nombre de la base test por default,
    usuario password: admin / admin
    language: Spanish, Country: Argentina, Load data = True y le instala el
    modulo default.
